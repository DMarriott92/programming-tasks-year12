# Week 12 — Advanced OOP: Inheritance & Polymorphism

This week extends your OOP knowledge to cover polymorphism and more complex
inheritance hierarchies. These are the OOP concepts most likely to appear in
the AQA A-Level programming paper.

## Tasks

| File | Task | Key Concept |
|------|------|-------------|
| `01_shape_hierarchy.py` | Build a shape hierarchy with `Shape`, `Circle`, `Rectangle`, and `Triangle` classes, each with an `area()` method | Polymorphism, method overriding |
| `02_rpg_battle_system.py` | Design an RPG battle system with a `Character` base class and subclasses like `Warrior` and `Mage` | Inheritance, polymorphism |
| `03_vehicle_rental_system.py` | Model a vehicle rental system with shared and specialised behaviour across vehicle types | Inheritance hierarchies, encapsulation |

## Key Concepts This Week
- **Polymorphism** — the same method name behaves differently depending on the object
- **Method overriding** — a subclass provides its own version of a parent method
- **`super()`** — calling the parent class constructor or method from a subclass
