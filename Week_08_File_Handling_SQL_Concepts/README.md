# Week 08 — File Handling & SQL Concepts

This week bridges file handling and databases, introducing SQL for the first time.
You will work with SQLite — a lightweight database that requires no server setup —
using Python's built-in `sqlite3` module.

## Tasks

| File | Task | Key Concept |
|------|------|-------------|
| `01_grade_writer.py` | Read student scores and write grade results to a new file | File I/O, data processing |
| `02_CSV_Product_Loader.py` | Load product data from a CSV file into a SQLite database | CSV → SQL, `sqlite3` |
| `03_SQL_Single_Table_Intro.py` | Create a single-table SQLite database and perform basic queries | SQL, `CREATE`, `INSERT`, `SELECT` |

## Getting Started with SQLite
No installation needed — `sqlite3` is part of Python's standard library. You can
view your `.db` files using the **SQLite VS Code extension** (already in the
recommended extensions for this repo).
