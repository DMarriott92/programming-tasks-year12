# Week 19 — Client-Server Programming & Web APIs

This week introduces networked programming — making HTTP requests to web APIs,
building a simple server with Flask, and implementing client-server communication.
This is the most practical week in terms of real-world application development.

## Tasks

| File | Task | Key Concept |
|------|------|-------------|
| `01_API_client.py` | Make requests to a public web API and process the JSON response | HTTP requests, `requests` library, JSON |
| `02_flash_server.py` | Build a simple web server using Flask with basic routes | Flask, HTTP, client-server model |
| `03_clientserver_messaging.py` | Implement a basic client-server messaging system using sockets | Sockets, networking |

## Dependencies
This week requires additional libraries. Make sure you have run:
```bash
pip install -r requirements.txt
```
The key libraries are `requests` (for API calls) and `flask` (for the server).
