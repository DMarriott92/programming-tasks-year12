# Week 02 — GCSE Basics

This week revisits and consolidates GCSE-level programming concepts, extending them
with more rigorous logic and input validation. If Week 01 felt comfortable, these
tasks should feel like familiar territory approached with more care.

## Tasks

| File | Task | Key Concept |
|------|------|-------------|
| `01_Grade_Calculation.py` | Calculate and classify a student grade from a score | Selection, conditionals |
| `02_Times_Tables.py` | Generate and display a times table | Nested loops |
| `03_queue_simulation.py` | Simulate a simple queue with join/leave operations | Lists as queues |
| `04_Password_Strength.py` | Assess the strength of a password against criteria | String validation |
| `05_Shopping_List.py` | Manage a shopping list with add/remove/display | List manipulation |
| `06_Simple_Login.py` | Implement a basic login system with attempt limiting | Loops, conditionals |

## AQA Spec Links
- Selection and iteration
- Lists and list operations
- String handling and validation
- Subroutines
