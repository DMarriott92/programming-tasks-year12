## Week 23–26 — Final Major Project

```markdown
# Weeks 23–26 — Final Major Project

The final major project is your opportunity to design and build a substantial
program of your own choosing. It should demonstrate the breadth of what you have
learned this year and serve as a portfolio piece going into Year 13.

## Requirements

Your project must include:
- [ ] Object-oriented design with at least two classes
- [ ] Persistent data storage — SQLite database or file-backed
- [ ] Input validation and exception handling throughout
- [ ] A suite of pytest tests covering your core functions
- [ ] A `README.md` inside this folder explaining what your project does and how to run it

## Suggested Project Ideas
- A library management system
- A student grade tracker with reporting
- A quiz application with a question bank stored in a database
- A text-based RPG with save/load functionality
- A personal finance tracker

## Submission
Push your completed project to your fork before the deadline. Your teacher will
review your repo directly — make sure your code runs cleanly from a fresh clone
and all tests pass.
```
