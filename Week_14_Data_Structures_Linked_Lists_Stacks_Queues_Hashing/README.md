# Week 14 — Data Structures: Linked Lists, Stacks, Queues & Hashing

This week implements the abstract data structures from the AQA theory paper in
Python. You are building these from scratch using classes — do not use Python's
built-in `deque` or similar for the core implementations.

## Tasks

| File | Task | Key Concept |
|------|------|-------------|
| `01_linked_list.py` | Implement a singly linked list with append, delete, and traverse operations | Linked lists, pointers/references |
| `02_stack_and_queue.py` | Implement a stack (LIFO) and a queue (FIFO) | Stacks, queues |
| `02_stack_queue_manual.py` | Implement stack and queue manually using arrays rather than classes | Array-based data structures |
| `03_hash_table.py` | Implement a hash table with a simple hash function and collision handling | Hashing, hash tables |

## Key Vocabulary
- **Stack** — Last In, First Out (LIFO) — `push` and `pop`
- **Queue** — First In, First Out (FIFO) — `enqueue` and `dequeue`
- **Linked List** — nodes connected by pointers, no fixed size
- **Hash Table** — key-value store using a hash function to determine position

## AQA Spec Links
- Stacks and queues
- Linked lists
- Hash tables and hashing
