# Week 07 — OOP Fundamentals

This week introduces Object-Oriented Programming. You will design and implement
classes with attributes and methods, then extend them using inheritance. OOP is
a major topic in both the AQA theory and programming papers.

## Tasks

| File | Task | Key Concept |
|------|------|-------------|
| `01_BankAccount_Class.py` | Build a `BankAccount` class with deposit, withdrawal, and balance methods | Classes, encapsulation |
| `02_Library_Item_Inheritance.py` | Create a `LibraryItem` base class with `Book` and `DVD` subclasses | Inheritance |
| `03_Queue_Simulation_OOP.py` | Reimplement the Week 02 queue simulation using a class | OOP design, encapsulation |

## Key OOP Concepts This Week
- **Class** — a blueprint for an object
- **Object** — an instance of a class
- **Attribute** — data stored inside an object
- **Method** — a function that belongs to a class
- **Inheritance** — a subclass that extends a parent class
