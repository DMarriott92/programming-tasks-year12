# Weeks 20–22 — Integration Project

Over these three weeks you will build a larger, integrated application that
combines the skills developed throughout the year. The projects here bring
together OOP, file handling, SQL, and modular design in a single coherent program.

## Projects

| File | Project | Key Concepts Combined |
|------|---------|----------------------|
| `01_restaurant_ordering.py` | A restaurant ordering system with a menu, order management, and persistent storage | OOP, SQL, file handling, menus |
| `02_scheduling_optimiser.py` | A scheduling optimiser that assigns resources to time slots without conflicts | Algorithms, decomposition, OOP |

## Expectations for These Weeks
- Code should be well-structured and modular — functions and classes where appropriate
- Programs should handle invalid input gracefully using exception handling
- Data should be persisted — either to files or a SQLite database
- You should write at least a basic set of tests for your core functions
