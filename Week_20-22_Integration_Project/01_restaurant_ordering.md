# Restaurant Ordering System
Ok you've reached week 20, well done. Now to combine all of the skills above together.
You are going to create a basic Restaurant Ordering system, you will need to create a project folder here.
You are going to allow a user to order a number of items from a menu, the server side will be updated and simulate (in a basic way) the items being made, stock being reduced and order being stored.

Combined skills:
- OOP
- File Storage
- Simple scheduling
- menus
- pricing calculations
