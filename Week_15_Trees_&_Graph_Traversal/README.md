# Week 15 — Trees & Graph Traversal

This week implements tree structures and the two fundamental graph traversal
algorithms — BFS and DFS. These are heavily examined in the AQA theory paper
and are worth understanding thoroughly.

## Tasks

| File | Task | Key Concept |
|------|------|-------------|
| `01_binary_tree_traversal.py` | Build a binary tree and implement in-order, pre-order, and post-order traversal | Binary trees, tree traversal |
| `02_bfs_traversal.py` | Implement Breadth-First Search on a graph | BFS, queues |
| `03_dfs_cycle_detection.py` | Implement Depth-First Search and use it to detect cycles in a graph | DFS, stacks, cycle detection |

## BFS vs DFS
| | BFS | DFS |
|---|---|---|
| Uses | Queue | Stack |
| Explores | Level by level | As deep as possible first |
| Good for | Shortest path | Cycle detection, topological sort |
